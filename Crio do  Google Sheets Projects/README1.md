
### 📗 1. Excel_Practice_Arithmetic_Functions.xlsx

> **Topic:** Arithmetic Functions & Descriptive Data Analysis

This file challenges you to perform a **complete salary data analysis** using Excel's built-in arithmetic and statistical functions.

**🔧 What You'll Practice:**
- Calculating **HRA at 45% of Basic Salary** using percentage formulas
- Computing **Total Salary** using `SUM()`
- Finding **Average Salary** using `AVERAGE()`
- Building a **Descriptive Analysis Summary** section
- Applying formulas across multiple employee records

**📌 Key Functions Used:**
```excel
=SUM()         → Total of a range
=AVERAGE()     → Mean value
=MAX()         → Highest value
=MIN()         → Lowest value
= A1 * 45%     → Percentage calculation
```

**💡 Real-World Scenario:** You are an HR analyst preparing a salary summary report for management.
