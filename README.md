# HI Everyone
## 🌟 About This Repository

Welcome to **Excel Practice Problems** — a hands-on learning repository  🎯

Instead of just reading theory, this repo gives you **actual `.xlsx` practice files** with problem statements, datasets, and solution sheets — so you **learn by doing**.

Each file covers a distinct Excel concept, progressively building your skills from basic arithmetic formulas to advanced conditional logic and data validation.
